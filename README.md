# 🌦️ WeatherRain

Простое веб-приложение для получения текущей погоды по названию города.

## 🚀 Функциональность

- Поиск погоды по городу
- Отображение температуры
- Отображение "ощущается как"
- Влажность воздуха
- Скорость ветра
- Осадки (если есть)
- Обработка ошибок (например, неверный город)
- Loading state при загрузке данных

## 🛠️ Стек технологий

- JavaScript (ES6+)
- HTML / CSS
- Vite
- REST API (OpenWeatherMap)
- Fetch / Async-Await

## Установка и запуск

```bash
git clone https://github.com/GreaterBerg/WeatherRain
cd WeatherRain
npm install
npm run dev
